Ajouter/Modifier un noeud- Extrapolation d'un noeud

But :L'onglet d'extrapolation permet aux utilisateurs de prévoir les valeurs futures des nœuds en extrapolant à partir des valeurs passées. Cette fonctionnalité n'est disponible que sur les nœuds numériques et est similaire au 'Extrapolation' Méthodes d'extrapolation disponibles dans QAT

Utilisation de cet onglet:

  1. Dans le tableau, entrer n'importe quelle valeur réelle ainsi que les taux de rapportage pour vos données historiques. À partir de cette saisie, QAT calculera les valeurs historiques ajustées.
  2. S'il manque des données, utiliser le bouton vert Interpoler pour compléter les données manquantes.
  3. (facultatif) En haut de l'écran, sélectionner les méthodes de prévision que vous souhaitez afficher et mettre à jour les paramètres d'extrapolation.
  4. Cliquez sur « Extrapoler ». Toutes les méthodes de prévisions sélectionnées s'afficheront dans le tableau principal et les graphiques.

    5. Note: mois de données sur la consommation réelle. Les valeurs minimales nécessaires pour les différentes fonctionnalités sont ci-dessous:
    * LET, Holt-Winters: Nécessite au moins 24 mois de données de consommation réelle
     * ARIMA: Saisonnalité : Au moins 13 mois de données historiques requises. Sans saisonnalité : Au moins 2 mois de données historiques requises
    * Moyenne mobile

  5. (facultatif) Dans le tableau, faire n'importe quelles modifications manuelles (+/-). Ces modifications sont ajoutées ou soustraites aux valeurs extrapolées.
  6. Après avoir examiné le tableau principal, le tableau des erreurs et les graphiques, sélectionner la méthode de prévision souhaitée en bas de l'écran et cliquez sur Enregistrer.
