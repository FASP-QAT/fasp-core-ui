Entrada e Ajuste de Dados

Objetivo : Permita que os usuários preparem dados históricos de consumo antes de avançar 'Extrapolação' tela. O consumo real histórico pode ser quantidades vendidas, quantidades dispensadas ao usuário ou quantidades emitidas por instalações de armazenamento. Nesta tela, os usuários podem ajustar os dados históricos de consumo que foram inseridos manualmente nesta tela ou que foram importados de um programa de plano de abastecimento QAT (se desejar importar, prossiga primeiro para o 'Importar do plano de abastecimento QAT' screen).

Usando esta tela :

  1. Selecione o programa de previsão e o período de revisão desejado. Por padrão, o QAT pré-seleciona um intervalo de datas de 36 meses antes do início do período de previsão. Uma tabela não editável e todos os produtos relacionados a este programa de previsão são exibidos, juntamente com seus dados de consumo ajustados, caso você tenha inserido esses dados anteriormente; caso contrário, você verá células amarelas indicando que nenhum dado de consumo foi inserido. Por padrão, os produtos e o consumo são mostrados em unidades de planejamento. O usuário também pode expandir cada linha da tabela para exibir o consumo por região, se estiver gerenciando um programa multirregional.
  2. Na tabela superior, clique em um produto para exibir a tabela de dados detalhada desse produto. A tabela de dados detalhada terá uma seção para cada região em seu programa.
  3. A tabela de dados detalhada permite que os usuários adicionem, editem, ajustem ou excluam registros históricos de consumo.
    1. Se desejar inserir dados manualmente, insira o consumo histórico para cada região e mês na linha "Consumo real".
    2. Se você importou dados do QAT, verá seu consumo histórico importado na linha "Consumo real".
    3. Por padrão, supõe-se que os dados sejam inseridos na Unidade de planejamento. No entanto, o usuário pode especificar a unidade apropriada para os dados detalhados clicando no link alterar sob o nome do produto e no pop-up subsequente, optando por inserir dados usando a unidade de planejamento, as quantidades da unidade de previsão (aplicando um domínio -fator de conversão gerenciado) ou outra unidade inserida pelo usuário (e fator de conversão).
  4. Existem três maneiras de ajustar os dados:
    1. Ajustar para subnotificação: O valor padrão é 100% de relatórios todos os meses. O usuário pode alterar isso para o valor correto. A QAT calculará o consumo ajustado devido à subnotificação. O cálculo assume que as instalações que não relataram nenhum consumo tiveram a mesma taxa de consumo das que o fizeram.
    2. Ajuste para falta de estoque: Para dados importados, o número de dias de falta de estoque é obtido do programa de plano de fornecimento QAT, se os dados forem coletados. O valor padrão para dias de falta de estoque é zero dias (produto assumido sempre em estoque). O usuário pode alterar isso para o valor correto. O cálculo pressupõe que os dias em que o produto ficou sem estoque teriam a mesma taxa de consumo que o restante do tempo em que o produto ficou em estoque. O valor padrão para o número de dias em um mês é baseado nos dias do calendário, mas os usuários podem ajustar o número de dias usados para o cálculo de falta de estoque em 'Atualizar configurações de versão'.

      3. Taxa de Falta de Estoque = Esgotado (dias)/ (Nº de dias no mês).
      Consumo Ajustado = Consumo real / Taxa de Relatório / (1 - Taxa de Esgotamento)

      Por exemplo, se para um determinado mês, um produto teve um consumo de 1.000 unidades, ficou fora de estoque por 5 dos 31 dias do mês e a taxa de relatório foi de 98%:
      Taxa de Falta de Estoque = 5 days stocked out /31 days in a month = 16.1%.
      Consumo Ajustado = 1,000 units / 98% Reporting / (1 - 16.1%) = 1,217

    3. Ajustando para valores ausentes: Clique no botão verde "Interpolar" no meio direito da tela para procurar períodos em que o valor de consumo está em branco e substituí-los por um valor interpolado. QAT interpola encontrando os valores mais próximos em cada lado (antes ou depois do branco), calcula a linha reta entre eles e usa essa fórmula de linha reta para calcular o valor do(s) branco(s). Observe que o QAT não fará a interpolação para meses em que o consumo real for zero. O QAT apenas interpolará se houver pelo menos um ponto de dados antes e um ponto de dados após o(s) valor(es) em branco.
