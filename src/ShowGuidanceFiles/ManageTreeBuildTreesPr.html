Gerenciar Árvore - Construir Árvores

Objetivo Permita que os usuários gerenciem e construam árvores e cenários de previsão, para quaisquer previsões de não consumo (demográficas, morbidade, serviços, etc). Observe que mais orientações estão disponíveis após clicar em qualquer nó específico.

Usando esta tela
Navegação:

  • Comece selecionando o Programa e a Árvore nos menus suspensos. Observe que apenas programas baixados estão disponíveis. Clique no ícone de engrenagem para mostrar/ocultar ou editar os detalhes da árvore (método de previsão, nome da árvore e região).
  • Veja a seção abaixo para Cenários.
  • Os valores dos nós da árvore são exibidos na data mostrada no Data de exibição. Altere esta data para ver valores em outros meses.
  • Os usuários também podem editar a árvore na tela Tree Table, que permite a edição em lote dos valores dos nós, mas não permite quaisquer edições na estrutura da árvore ou alterações ao longo do tempo.
  • Caixas de seleção “Ocultar” - use essas caixas de seleção para ocultar certos elementos da árvore para uma exibição mais limpa.
  • Calcular automaticamente - Se o QAT estiver lento durante a edição da sua árvore, desmarque a caixa de cálculo automático para interromper os cálculos. Para atualizar os cálculos, marque novamente a caixa de seleção 'calcular automaticamente' ou clique no ícone recalcular.
  • Clique nos Níveis da árvore (caixas retangulares azuis escuras à esquerda da árvore) para:
    • Nomeie os níveis de uma árvore. Isso tornará a árvore mais fácil de ler e será usada nos menus suspensos e no eixo da tela Validação de Modelagem.
    • Reordene os galhos da sua árvore conforme desejado usando as setas para cima e para baixo para indicar a posição dos galhos da esquerda para a direita. Se a lista for longa, use o menu suspenso “ver filhos de” para filtrar os nós.

Construindo a árvore:A árvore de previsão é construída de cima para baixo, usando diferentes tipos de “nós”. Consulte os tipos de nó e ações de nó abaixo. Cada árvore de previsão deve incluir um ou mais nós de unidade de planejamento, que formam a saída de previsão, para que essa árvore fique disponível na tela "Comparar e selecionar". Cada Nó de Unidade de Planejamento deve derivar de um Nó de Unidade de Previsão.

Ações do nó:

  • Clique em um nó para abri-lo para edição ou visualização.
  • Passe o mouse sobre um nó para visualizar as notas do nó sem abri-lo.
  • Copiar / Mover: Copia ou move o nó selecionado e todos os seus filhos. O usuário pode selecionar o destino.
  • Excluir: Exclui o nó selecionado e todos os seus filhos.
  • Adicionar modelo de filial: Adiciona um modelo de árvore como uma ramificação abaixo do nó escolhido.
  • Adicionar: Adiciona um nó filho ao nó selecionado.
  • / Recolher/Expandir: Recolhe/Expande o nó atual e todos os seus filhos.
    • No modo recolhido, os nós serão representados por pontos. Passe o mouse sobre qualquer ponto/nó e clique uma vez para expandir esse nó e todos os seus filhos.
    • “Recolher árvore” caixa de seleção na parte superior da tela que recolhe todos os nós, exceto o nó superior.

Mudanças ao longo do tempo

Embora a estrutura da árvore permaneça constante ao longo do tempo, as porcentagens e os valores podem mudar com o tempo. Use o menu suspenso de data para visualizar a árvore a qualquer momento mês. As três funcionalidades abaixo estão disponíveis em cada nó e permitem o usuário controla como os nós mudam ao longo do tempo:

  • Modelagem: Permite que o usuário especifique a taxa exata de mudança
  • Transferir: Permite que os usuários vinculem dois nós - de modo que a diminuição do nó de origem esteja vinculada ao aumento do nó de destino. Útil para transições.
  • Extrapolação: Permite que os usuários usem dados históricos para extrapolar alterações futuras

Usando cenários :
Cenários são usados para modelar valores diferentes para a mesma árvore. Para adicionar, editar ou excluir um cenário, use o ícone ao lado da lista suspensa do cenário. Use a lista suspensa de cenários para selecionar qual cenário visualizar e editar.
