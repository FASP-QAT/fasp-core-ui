Saisie de données et ajustement

But : Permettre aux utilisateurs de préparer les données historiques de consommation avant de passer à l' 'Extrapolation' écran. La consommation réelle historique peut être des quantités vendues, des quantités distribuées à l'utilisateur ou des quantités émises par les installations de stockage. Sur cet écran, les utilisateurs peuvent ajuster les données historiques de consommation qui ont été saisies manuellement sur cet écran ou qui ont été importées d'un programme de plan dapprovisionnement QAT (si vous souhaitez importer, veuillez dabord passer à la 'Importation du module Planification des approvisionnements de QAT' screen).

Utilisation de cet écran:

  1. Sélectionner le programme de prévision et la période souhaitée de revue. Par défaut, QAT présélectionne une période de 36 mois avant le début de la période de prévision. Un tableau non modifiable et tous les produits liés à ce programme de prévision sont affichés, ainsi que leurs données de consommation si vous avez préalablement saisi ces données ; sinon, vous verrez des cellules jaunes indiquant qu'aucune donnée de consommation n'a été saisie. Par défaut, les produits et la consommation sont affichés en unités de planification. L'utilisateur peut également développer chaque ligne du tableau pour afficher la consommation par région, s'il gère un programme multi-régions.
  2. Dans le tableau du haut, cliquer sur un produit pour afficher le tableau de données détaillé de ce produit. Le tableau de données détaillées comportera une section pour chaque région de votre programme.
  3. Le tableau de données détaillées permet aux utilisateurs d'ajouter, de modifier, d'ajuster ou de supprimer des enregistrements de consommation historiques.
    1. Si vous souhaitez saisir les données manuellement, saisissez la consommation historique pour chaque région et mois dans la ligne « Consommation réelle ».
    2. Si vous avez importé des données de QAT, vous verrez votre consommation historique importée dans la ligne « Consommation réelle ».
    3. Par défaut, les données sont supposées être saisies dans l'unité de planification. Cependant, l'utilisateur peut spécifier l'unité appropriée pour les données détaillées en cliquant sur le lien Modifier sous le nom du produit et dans la fenêtre contextuelle suivante, en choisissant d'entrer des données en utilisant l'unité de planification, les quantités d'unité de prévision (en appliquant un facteur de conversion géré au niveau du domaine) ou une autre unité saisie par l'utilisateur (et facteur de conversion).
  4. Il existe trois façons d'ajuster les données:
    1. Ajuster en cas de sous-rapportage: La valeur par défaut est 100 % de rapportage chaque mois. L'utilisateur peut le remplacer par la valeur correcte. QAT calculera la consommation ajustée en raison du sous-rapportage. Le calcul suppose que les installations qui n'ont déclaré aucune consommation avaient le même taux de consommation que celles qui l'ont fait.
    2. Ajuster pour les ruptures de stock: Pour les données importées, le nombre de jours de rupture de stock est extrait du programme de plan d'approvisionnement QAT, si les données sont collectées. La valeur par défaut pour les jours de rupture de stock est de zéro jour (produit supposé toujours en stock). L'utilisateur peut le remplacer par la valeur correcte. Le calcul suppose que les jours où le produit était en rupture de stock auraient eu le même taux de consommation que le reste du temps où le produit était en stock. La valeur par défaut du nombre de jours dans un mois est basée sur les jours calendaires, mais les utilisateurs peuvent ajuster le nombre de jours utilisés pour le calcul de la rupture de stock dans 'Mettre à jour les paramètres de version'.

      3. Taux de rupture de stock = Rupture de stock (en jours)/ (Nombre de jours dans le mois).
      Consommation ajustée = Consommation réelle / Taux de rapportage / (1 - Taux de rupture de stock)

      Par exemple, si pour un mois donné, un produit avait une consommation de 1 000 unités, était en rupture de stock pendant 5 jours sur 31 dans le mois et le taux de rapportage était de 98 %:
      Taux de rupture de stock = 5 days stocked out /31 days in a month = 16.1%.
      Consommation ajustée = 1,000 units / 98% Reporting / (1 - 16.1%) = 1,217

    3. Ajustement des valeurs manquantes: Cliquez sur le bouton vert « Interpoler » au milieu à droite de l'écran pour rechercher les périodes où la valeur de consommation est vide et les remplacer par une valeur interpolée. QAT interpole en trouvant les valeurs les plus proches de chaque côté (avant ou après le blanc), calcule la ligne droite entre eux et utilise cette formule linéaire pour calculer la valeur du ou des blancs. Notez que QAT n'effectuera pas d'interpolation pour les mois où la consommation réelle est de zéro. QAT n'interpolera que s'il y a au moins un point de données avant et un point de données après la ou les champs vides.

    4. Utiliser le graphique sous le tableau des données détaillées pour afficher les données ajustées (Remarque : le graphique ne sera mis à jour qu'après avoir cliqué sur "Soumettre")

  6. Répétez les étapes 2 à 4 pour chaque unité de planification
