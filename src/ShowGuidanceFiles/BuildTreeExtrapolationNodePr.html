Adicionar/editar nó - nó de extrapolação

Objetivo :A guia de extrapolação permite que os usuários prevejam valores futuros de nós extrapolando valores passados. Esta funcionalidade está disponível apenas em nós numéricos e é semelhante ao 'Extrapolação' Métodos de extrapolação disponíveis no QAT

Usando esta guia:

  1. Na tabela, insira quaisquer valores reais e taxas de relatório para seus dados anteriores. A partir dessa entrada, o QAT calculará os valores históricos ajustados.
  2. Se houver dados ausentes, use o botão verde Interpolar para preencher os dados ausentes.
  3. (opcional) Na parte superior da tela, selecione quais métodos de previsão você deseja exibir e atualize os parâmetros de extrapolação.
  4. Clique em “Extrapolar”. Todas as previsões selecionadas serão exibidas na tabela principal e nos gráficos.

    5. Observação: meses de dados de consumo real. Os valores mínimos necessários para os vários recursos estão abaixo:
    * TES, Holt-Invernos: Precisa de pelo menos 24 meses de dados de consumo real
     * ARIMA: Sazonalidade : São necessários pelo menos 13 meses de dados históricos.Sem sazonalidade: São necessários pelo menos 2 meses de dados históricos
    * Média móvel

  5. (opcional) Na tabela, adicione quaisquer alterações manuais (+/-). Essas alterações são adicionadas ou subtraídas em cima dos valores extrapolados.
  6. Após revisar a tabela principal, a tabela de erros e os gráficos, selecione o método de previsão desejado na parte inferior da tela e clique em “Salvar”.
