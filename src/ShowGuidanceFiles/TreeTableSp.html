Consejos para casos de uso específicos:

  • Tipo de nodo de previsión : Seleccione discreto si este producto se usará por tiempo limitado y continuo si se usará indefinidamente.
  • ¿Uso del producto retrasado o escalonado? A veces, el consumo del producto se retrasa en relación a los otros niveles superiores del árbol. en lo pertinente Unidad de previsión nodo, utilice el Retraso campo para indicar este retraso.
    • Por ejemplo, si el uso del producto comenzará 2 meses después de las fechas del nodo principal, ingrese 2 en este campo.
    • Este campo también se puede usar cuando el producto cambia con el tiempo, por ejemplo, si las unidades de pronóstico A, B y C se usan en secesión durante dos meses a la vez, puede configurar su árbol con la Unidad de pronóstico A (discreta para 2 meses). , lag=0), Unidad de Pronóstico B (discreta por 2 meses, lag=2), Unidad de Pronóstico C (discreta por 2 meses, lag = 4).
