Gestion des coûts de fret, des délais et des prix des unités de planification

Cette écran vous permet de gérer des détails spécifiques tels que les coûts de fret, les délais et les prix des unités de planification pour les agents d'approvisionnement et les unités de planification dans votre programme de planification des approvisionnements.

Ajout ou modification de lignes

Pour ajouter une ligne, suivez ces étapes :

  • Programme, Unité de planification et Agent d'approvisionnement : Remplissez ces champs.
  • Création d'une combinaison Agent d'approvisionnement/Unité de planification : Sélectionnez des options à la fois dans les colonnes unité de planification et agent d'approvisionnement.
  • Ligne spécifique à l'agent d'approvisionnement sans lien avec une unité de planification : Choisissez ‘Tous’ dans la colonne unité de planification.
  • Coûts de fret, délais et prix unitaire : Au moins une des colonnes restantes doit avoir une valeur numérique.
Contraintes et règles
  • Une seule ligne est autorisée par combinaison programme/unité de planification/agent d'approvisionnement.
  • Si une ligne d'agent d'approvisionnement inclut ‘Tous’ les unités de planification, la colonne prix unitaire sera non modifiable.
Utilisation dans le module de planification des approvisionnements

Ces détails auront un impact sur votre module de planification des approvisionnements comme suit :

  • Pourcentages de fret : Priorisés par les paramètres programme/agent d'approvisionnement/unité de planification puis les paramètres du programme.
  • Délais : Priorisés par les paramètres programme/agent d'approvisionnement/unité de planification, puis les paramètres de l'agent d'approvisionnement au niveau du domaine et enfin les paramètres du programme.
  • Prix unitaire : Priorisés par les paramètres programme/agent d'approvisionnement/unité de planification, puis les paramètres de l'agent d'approvisionnement au niveau du domaine et enfin les paramètres du programme.
