Gerenciar Árvore - Lista de Árvores

Objetivo Permita que os usuários

  1. Ver uma lista de suas árvores existentes
  2. Edite uma árvore existente clicando em qualquer linha
  3. Excluir ou duplicar árvores existentes clicando com o botão direito em uma linha
  4. Adicione uma nova árvore a um programa carregado clicando no menu suspenso Adicionar árvore no canto superior direito da tela. Novas árvores podem ser construídas:
    • manualmente - selecione Adicionar árvore
    • de um modelo de árvore - selecione o nome do modelo desejado.

Observação: os visualizadores de previsão podem visualizar qualquer árvore que tenha sido confirmada no servidor e à qual tenham acesso; no entanto, apenas usuários/administradores de previsão poderão criar e editar/atualizar árvores.

Usando esta tela :

  • Um programa de previsão deve primeiro ser carregado para construir uma árvore (manualmente ou a partir de um modelo de árvore).
  • Antes de construir e editar uma árvore, primeiro adicione as unidades de planejamento do programa de previsão na Atualizar Unidade de Planejamento tela antes de construir
  • Construir uma árvore semelhante a uma árvore existente? Duplique uma árvore existente clicando com o botão direito do mouse em uma linha e selecionando “duplicar” editar. Se você quiser manter a estrutura da árvore constante e alterar apenas os números, construa apenas uma árvore e use o recurso de cenário.
  • Envie um tíquete de HelpDesk se houver um modelo ausente que beneficie a comunidade QAT.
