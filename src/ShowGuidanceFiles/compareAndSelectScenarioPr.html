Comparar e selecionar:

Objetivo : Permita que os usuários comparem todas as previsões disponíveis (de árvore e métodos de consumo) e selecionem sua previsão final. Nesta tela, os usuários selecionam suas previsões uma unidade de planejamento e região por vez. Para selecionar previsões em várias unidades de planejamento e regiões, use o

Usando esta tela

  1. Verifique se todas as previsões esperadas aparecem na tabela Comparar e selecionar. Para adicionar previsões, navegue até o Gerenciar árvore tela para construir uma previsão de árvore ou o Extrapolação tela para construir uma previsão baseada no consumo. o símbolo indica que não há quantidade prevista disponível; a linha inteira ficará cinza e não editável, pois não há nada para exibir ou selecionar.
  2. Revise as previsões disponíveis usando as seguintes informações na tela:
    • Erro de previsão (%): esse erro é calculado usando o cálculo do Erro de Porcentagem Absoluta Ponderada (WAPE). Se o erro de previsão estiver destacado em texto verde, esta previsão tem o menor erro de previsão entre as previsões disponíveis. Exemplo de cálculo WAPE usando a janela de tempo padrão de 6 meses:
      Se 6 meses de dados não estiverem disponíveis, o QAT utilizará quantos meses estiverem disponíveis e os indicará nesta coluna.
    • Comparar com a previsão de consumo: QAT compara Previsões de Consumo e Previsões de Árvore disponíveis. Para qualquer Previsão de Árvore, o QAT sinalizará a porcentagem acima da Previsão de Consumo mais alta ou abaixo da Previsão de Consumo mais baixa. A comparação será destacada em texto vermelho se estiver fora das porcentagens de limite definidas pelo usuário no
      Configuración de versión Assumindo dados confiáveis de consumo real, esta comparação ajuda os usuários a determinar se suas Previsões de Árvore são realistas.
    • Gráfico: Compare visualmente as diferentes previsões. A previsão selecionada aparecerá negrito.
    • Tabela de dados tabulares: Compare os dados entre as previsões lado a lado clicando no botão Mostrar dados abaixo do gráfico. Algum negrito/roxo em itálico em negrito os dados estão dentro do período de previsão.
  3. Selecione a previsão final na tabela Comparar e Selecionar. Repita as etapas 1 a 3 para cada unidade de planejamento e região. Uma vez concluído, continue em frente para o Previsão Mensal para verificar todas as unidades de planejamento juntas.

Dicas sobre como usar os dados gráficos e tabulares:

  • Par défaut, QAT affichera toutes les prévisions disponibles par unité de planification et tous les chiffres réels saisis ou importés du module planification des approvisionnements QAT ; cependant, un utilisateur peut désélectionner la case « Afficher ? » pour toutes les prévisions dans le tableau du haut s'ils ne souhaitent pas les voir sur le graphique.
  • Un utilisateur peut afficher le graphique en unité de prévision, unité d'équivalence et pour une période de temps spécifique. Si un utilisateur choisit d'afficher uniquement la période de prévision, le graphique n'affichera que la période de temps que l'utilisateur a choisie comme période de prévision dans le Paramètres de version screen.
