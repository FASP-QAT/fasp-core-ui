Comparar e selecionar

Objetivo : Permitir que os usuários comparem todas as previsões disponíveis (da árvore e métodos de consumo) e selecione sua previsão final. Nesta tela, os usuários selecione suas previsões, uma unidade de planejamento e região de cada vez. Para selecionar previsões em diversas unidades de planejamento e regiões, use o Resumo da previsão.

Usando esta tela :

  1. Verifique se todas as previsões esperadas aparecem na caixa Comparar e Selecionar mesa. Para adicionar previsões, navegue até Gerenciar árvore tela para construir um previsão de árvore ou Extrapolação da tela para construir uma base de consumo previsão. Quaisquer linhas que aparecem em cinza e não editáveis ​​indicam que há não há quantidade prevista disponível, pois não há dados para isso previsão específica.
  2. Revise as previsões disponíveis usando as seguintes informações na tela:
    • Erro de previsão (%): este erro é calculado usando o Cálculo do erro percentual absoluto ponderado (WAPE). Se o erro de previsão é destacado em texto verde, este previsão tem o menor erro de previsão dentre os disponíveis previsões. Exemplo de cálculo WAPE usando o padrão de 6 meses janela de tempo:
      Se 6 meses de dados forem não estiver disponível, o QAT utilizará quantos meses estiverem disponíveis e indique-o nesta coluna.
    • Comparar com a previsão de consumo: comparações QAT disponíveis Previsões de consumo e previsões de árvores. Para qualquer árvore Previsões, o QAT sinalizará a porcentagem acima do valor mais alto ou abaixo da menor Previsão de Consumo. A comparação será destacado em vermelho texto se estiver fora das porcentagens limite definido pelo usuário em
      Versão Tela de configurações. Assumindo um consumo real confiável dados, esta comparação ajuda os usuários a determinar se sua árvore As previsões são realistas.
    • Gráfico: Compare visualmente as diferentes previsões. A previsão selecionada aparecerá negrito.
    • Tabela de dados tabulares: compare os dados entre as previsões lado a lado clicando no botão “Mostrar dados” abaixo do gráfico. Qualquer negrito/negrito os dados roxos em itálico se enquadram no período de previsão.
  3. Selecione a previsão final na tabela Comparar e Selecionar. Se houver um aviso com o símbolo de preenchimento sólido próximo à Unidade de planejamento, não há previsão selecionada. Se houver uma caixa de seleção marcada com símbolo de preenchimento sólido próximo à unidade de planejamento, há uma previsão selecionada. Repita as etapas 1 a 3 para cada unidade de planejamento e Região até não mais Aviso com símbolos de preenchimento sólido permanecem. Depois de concluído, continue em frente para Mensalmente Previsão para verificar todas as unidades de planejamento juntas.

Nota: Os usuários podem selecionar várias previsões de árvores como previsão final, o que vai agregar as quantidades previstas mensalmente; no entanto, os usuários não podem selecionar múltiplo previsões de consumo ou uma combinação de previsões de consumo e árvores.

Tips on Using the Graph & Tabular Data:

  • Por padrão, o QAT exibirá todas as previsões disponíveis por unidade de planejamento e quaisquer valores reais inseridos ou importados do módulo QAT Supply Planning; no entanto, um usuário pode desmarcar a opção “Exibir?” caixa de seleção para qualquer previsão na tabela superior se não desejarem visualizá-lo no gráfico.
  • Um usuário pode visualizar o gráfico em Unidade de Planejamento, Unidade de Previsão, Unidade de Equivalência, e por um período de tempo determinado. Se um usuário optar por “Mostrar apenas período de previsão”, o gráfico exibirá apenas o período de horário que o usuário escolheu como período de previsão no Versão Tela de configurações.
  • Se a opção “Exibir previsões em meses com consumo real” estiver marcada, o gráfico calculará retroativamente as previsões de consumo para comparação com os reais.
  • Os usuários também podem optar por exibir o gráfico e a tabela de dados por Calendário Ano ou Ano Fiscal alternando o menu suspenso “Exibir por”. QAT irá preencher uma tabela de resumo do ano civil/ano fiscal abaixo do gráfico com quantidades agregadas anuais.
