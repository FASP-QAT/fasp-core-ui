Extrapolação

Objetivo :Permita que os usuários criem uma previsão identificando tendências e estações a partir de dados históricos de séries temporais inseridos pelo usuário no 'Entrada e Ajuste de Dados' tela. Várias opções de extrapolação estatística estão disponíveis e serão descritas abaixo. As extrapolações dos dados de consumo são feitas com base na unidade de planejamento por unidade de planejamento.

Usando esta tela:

1.Iniciando a extrapolação:

  • Antes de usar esta tela, certifique-se de ter concluído o 'Entrada e Ajuste de Dados' tela para cada unidade de planejamento e região que você deseja extrapolar
  • Use os filtros na parte superior desta tela para selecionar a unidade de planejamento, a região a ser extrapolada e o intervalo de datas a serem utilizados pelos dados históricos.
  • O usuário pode selecionar um ou vários dos métodos de extrapolação disponíveis no QAT, conforme apropriado para o padrão geral de consumo e dados disponíveis, como será discutido abaixo. Consulte as seções 2-4 abaixo para obter uma explicação dos métodos de extrapolação e como escolher os melhores métodos para sua previsão.
  • Observe que as alterações nessa tela são opcionais – o QAT aplica automaticamente os parâmetros de extrapolação padrão a todas as unidades de planejamento após a conclusão da tela Ajustes. O usuário então faz a seleção final da previsão no 'Compare e selecione a previsão' screen.
2.Métodos de extrapolação disponíveis no QAT

Os métodos de previsão no QAT são organizados do simples ao sofisticado.

  1. Média móvel: A média móvel é uma média que se move ao longo do tempo, eliminando dados mais antigos à medida que incorpora dados mais recentes. Para QAT calcular a média móvel, insira o número de meses no passado que você gostaria de usar no cálculo da média. O usuário pode selecionar qualquer número inteiro positivo para este campo. Digitar 5, por exemplo, significaria que a projeção para o próximo mês da série seria a média dos cinco meses anteriores. Este método é mais útil para previsões de curto prazo e é sensível às tendências. Não é apropriado para dados sazonais
  2. Semi-médias:semi-média estima tendências com base em duas metades de uma série. QAT divide os dados reais em duas partes iguais (metades) e a média aritmética dos valores de cada parte (metade) é calculada como os valores de y de dois pontos em uma linha. A inclinação da linha de tendência é determinada pela diferença entre esses valores de y ao longo do tempo, conforme definido pela diferença dos pontos médios das duas metades da série, ou valores de x, dos pontos. Este método é sensível às tendências e útil para previsões de curto e médio prazo, mas não é apropriado para dados sazonais.
  3. Regressão linear: A regressão linear modela a relação entre duas variáveis ajustando uma equação linear aos dados observados. Intervalo de confiança : entre 0% e 100% (exclusivo), por exemplo, nível de confiança de 90% indica que 90% dos possíveis pontos futuros devem cair dentro deste raio da previsão representada pela linha de regressão. Este método não é apropriado para dados sazonais.
  4. Suavização Exponencial Tripla (Holt-Winters): Nas estatísticas, diferentes tipos de suavização são usados ​​para filtrar o ruído para que possamos ver os padrões em um conjunto de dados de séries temporais com mais clareza. A suavização exponencial usa dados mais antigos com pesos decrescentes exponencialmente ao longo do tempo. Os parâmetros de suavização na suavização exponencial são definidos entre 0 e 1, com valores próximos a 1 favorecendo valores recentes e valores próximos a 0 favorecendo valores mais antigos. Os usuários que utilizam a suavização exponencial tripla podem escolher entre quatro parâmetros:
    • alfa, aplica-se ao nível ou linha de base do conjunto de dados. Valores alfa mais altos dão mais peso aos dados mais recentes
    • beta, determina quão fortemente as tendências recentes devem ser valorizadas em comparação com as tendências mais antigas
    • gama, reflete o componente sazonal da previsão. Sazonal geralmente se refere a padrões repetidos dentro de um ano. Quanto maior o gama, mais peso será aplicado ao componente sazonal mais recente dos dados.
    • Sazonalidade: No QAT, uma estação pode ser descrita como a duração esperada (em meses) de qualquer padrão repetitivo no consumo
    • Intervalo de confiança: entre 0% e 100% (exclusivo), por exemplo, nível de confiança de 90% indica que 90% dos pontos futuros devem cair dentro deste raio de previsão.
  5. ARIMA: A modelagem ARIMA permite que dois modelos estatísticos projetados para séries temporais estacionárias sejam integrados e aplicados a séries temporais não estacionárias, ou seja, séries temporais que possuem tendências ou estações. É frequentemente aplicado a previsões de curto prazo. Auto-regressivo significa que cada ponto na regressão é influenciado por seus valores anteriores e média móvel implica que cada ponto é uma média, uma combinação linear de um ou mais pontos adjacentes. Tanto a auto-regressão quanto a média móvel são integradas juntas para ajustar um melhor modelo para a série por meio de diferenciação, ou literalmente usando a diferença entre pontos em uma série temporal para a análise em vez dos valores brutos. Os modelos ARIMA têm três parâmetros:
    • p ou AR (ordem de atraso): o número de observações de atraso no modelo
    • d ou I (grau de diferenciação): o número de vezes que as observações brutas são diferenciadas. Esse valor normalmente é 1 (se houver tendência) ou 0 (sem tendência). Outros valores mais altos são possíveis, mas não esperados.
    • q ou MA (ordem da média móvel): o tamanho da janela de média móvel ou o número de observações diferenciadas a serem calculadas.
    • Os valores padrão de QAT para parâmetros ARIMA são p = 0, d = 1, q = 1.
    • Intervalo de confiança: entre 0% e 100% (exclusivo), por ex. O nível de confiança de 90% indica que 90% dos pontos futuros devem cair dentro desse raio da previsão.

3.Qual método de extrapolação devo usar?

Abaixo estão algumas considerações para selecionar um método de previsão. Os métodos de previsão no QAT são organizados do simples ao sofisticado. Em geral,

  • Modelos mais sofisticados são mais sensíveis a problemas nos dados
  • Se você tiver dados mais pobres (pontos de dados ausentes, taxas de relatórios variáveis, 12 meses de dados), métodos de previsão mais simples são recomendados

A escolha do método de extrapolação depende do padrão esperado nos dados. Alguns padrões típicos incluem:

  • Estacionário, onde o intervalo de valores observados ao longo do tempo paira em torno de uma média. Os modelos aplicados a esses conjuntos de dados podem incluir
    • Média móvel
    • ARIMA
  • Tendência sem componente sazonal, onde os valores observados têm tendência crescente ou decrescente. Os modelos aplicados a esses conjuntos de dados podem incluir
    • Semi-médias
    • Regressão Linear
  • Tendência e Sazonal, em que os valores observados em um conjunto de dados têm componentes de tendência e sazonais. Os modelos aplicados a esses conjuntos de dados podem incluir
    • Modelo ARIMA sazonal
    • Suavização Exponencial Tripla (Holt-Winters)
  • Sazonal sem tendência, onde os valores observados têm um componente sazonal, mas sem tendência. modelos aplicados a tais conjuntos de dados podem incluir
    • Suavização Exponencial Tripla (Holt-Winters)

Os modelos sugeridos aqui não são exaustivas nem exclusivas. QAT permite ao usuário aplicar uma variedade de métodos de extrapolação e, em seguida, compará-los usando os melhores ajustar ou prever métricas de erro.

A segunda etapa é considerar se os valores de previsão devem refletir de perto os padrões históricos em seus dados e, portanto, se você usará as métricas de erro para informar sua seleção. No entanto, ao optar por desconsiderar a métrica de erro, será importante documentar sua lógica para fazê-lo para informar discussões ou revisões de sua previsão e ajudar futuros analistas a apoiar suas decisões.

4.Como interpreto os erros?

O QAT calcula automaticamente as métricas de erro de previsão usando vários métodos. Esses incluem:

  • MAPE (erro percentual médio absoluto):pode ser interpretado como a porcentagem média diferença entre as previsões e os alvos pretendidos no conjunto de dados. Por exemplo, se MAPE é de 15%, então suas previsões estão em média 15% distantes dos valores reais.


  • WAPE (erro percentual absoluto ponderado): WAPE é uma métrica apropriada para quando o o conjunto de dados usado tem valores baixos ou intermitentes. Uma porcentagem WAPE de 5 significa que o a previsão estava errada em 5% em todo o conjunto de dados para um determinado período de avaliação. Dependendo No caso de uso, é recomendado escolher um modelo que forneça o menor valor WAPE.


  • RMSE (erro quadrático médio): O RMSE pode ser interpretado como o erro médio que o as previsões do modelo têm em comparação com o real, com peso extra adicionado a maiores erros de previsão. Geralmente, quanto mais próximo o RMSE estiver de zero, mais preciso será o modelo.


  • MSE (erro quadrático médio): MSE é a média agregada da diferença quadrática entre os fatos reais e as previsões. Grandes erros são destacados devido a esta quadratura. MSE é útil ao trabalhar em modelos onde grandes erros ocasionais devem ser minimizados. Geralmente, o mais próximo de 0, mais preciso é o modelo.


  • R 2 (a correlação quadrática entre a variável dependente e os valores ajustados): R 2 = 1 - [soma quadrada da regressão (SSR) / soma total dos quadrados (SST)], onde a soma ao quadrado dos resíduos da regressão (SSR) é a soma de seus resíduos ao quadrado, e o soma total dos quadrados (SST) é a soma da distância que os dados estão longe da média, todos ao quadrado. O valor R 2 é sempre um número entre 0 e 1. R 2 é uma medida do ajuste de um linha de regressão para uma série de observações históricas com valores mais próximos de 1 indicando uma correspondência mais forte entre os valores reais e previstos (modelo de regressão). Isso é não é uma medida de erro e não deve ser usada por si só para avaliar a adequação de uma previsão, mas em combinação com outras métricas de erro e o conhecimento do programa pelo usuário.


    • Além de R 2, quanto menor a pontuação, mais próximo o resultado do método de previsão se ajusta ao histórico dados. Em modelos onde se espera que os padrões históricos em um conjunto de dados sejam refletidos no futuro valores, um valor de erro baixo poderia ser usado para ajudar a selecionar um método de extrapolação preferido, e QAT destacará esses melhores ajustes em uma tabela. No entanto, se ocorrerem alterações substanciais que não se enquadrem no contexto histórico Se os dados forem antecipados, a extrapolação mais adequada pode não ser a previsão mais apropriada a ser selecionada.

    Para fazer a melhor seleção entre os métodos de extrapolação para o seu propósito, o mais importante é primeiro entender qual método de extrapolação é mais apropriado para o padrão esperado nos dados. Os métodos de previsão no QAT são organizados do simples ao sofisticado. Em geral,

    • Modelos mais sofisticados são mais sensíveis a problemas nos dados
    • Se você tiver dados mais pobres (pontos de dados ausentes, taxas de relatórios variáveis, 12 meses de dados), métodos de previsão mais simples são recomendados
      A segunda etapa é considerar se os valores de previsão devem refletir de perto os padrões históricos em seus dados e, portanto, se você usará as métricas de erro para informar sua seleção. No entanto, ao optar por desconsiderar a métrica de erro, será importante documentar sua lógica para fazê-lo para informar discussões ou revisões de sua previsão e ajudar futuros analistas a apoiar suas decisões.
